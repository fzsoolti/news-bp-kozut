# news-bp-kozut
To start the server:
// 1. install nodejs (min 18.0.0) https://nodejs.org/en/download/current
// 2. install mongodb server https://www.mongodb.com/try/download
// 3. terminal:
//              - npm i
//              - npm start (dev)
//              - npm run start:prod (prod)
//              - npm run debug (dev debug)
//              - npm run debug:prod (prod debug)

To start the client:
// 1. install nodejs (min 18.0.0) https://nodejs.org/en/download/current
// 2. terminal:
//              - npm i
//              - ng serve
