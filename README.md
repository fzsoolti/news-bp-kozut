# news-bp-kozut
To start the server:
// 1. install nodejs (min 18.0.0) https://nodejs.org/en/download/current
// 2. install mongodb server https://www.mongodb.com/try/download
// 3. terminal:
//                  - npm i
//(dev)             - npm start
//(prod)            - npm run start:prod 
//(dev debug)       - npm run debug 
//(prod debug)      - npm run debug:prod 

To start the client:
// 1. install nodejs (min 18.0.0) https://nodejs.org/en/download/current
// 2. terminal:
//              - npm i
//              - ng serve
